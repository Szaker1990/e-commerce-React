## E-Commerce React App

Live Demo [Click Here](https://szaker-clothing.herokuapp.com/).

## Technologies

- React
- Redux
- React Router
- Firebase Auth
- Firebase Firestore
- Styled Components
- Local Storage by Redux Persist
- React-create-app
- Stripe for Payments
- NPM


## How to run? 

#### `git clone https://github.com/Szaker1990/e-commerce-React.git`
#### `npm install`
#### `npm start`
